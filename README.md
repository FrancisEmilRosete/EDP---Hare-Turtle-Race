Hare and Turtle Race 

This Python script transforms the famous fable into a 
text-based simulation using the random module. Instead of a 
predetermined story, the code uses probability to determine 
when and how long the "Bunny" naps, while the "Turtle" 
maintains a slow, steady pace.
